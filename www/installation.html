How to Install in Windows system #back to top

Before you begin make sure that you have installed windows media player and whitelist video files in you antivirus software if any. In windows system, Tuteone videos are arranged in exe format. It requires a unique key/password to view the videos. When you open the video files, it asks for a play back password.

The playback passwords are provided by TuteOne. To get the password, copy the machine ID and send it to Tuteone via Whatsapp/Email (Only text machine ID is accepted , no images ) . We will reply back with your unique password.

Copy paste the given password in the popup dialogue box .To view each video for the first time you need to repeat this.

Tick the remember play back password dialogue box to avoid repeated password entry and then click on OK button.

Windows Trouble Shoot #back to installation menu #back to top

Can't open the .exe file #back to installation menu #back to top

  • 1. Reason - The player needs Windows Media Player to play the videos.
  • Solution - Install Windows Media Player.
  • 2. Reason - Anti Virus.
  • Solution - Disable Anti Virus / Exclude TuteOne video files from scanning (Whitelist)/Uninstall the Anti Virus
  • 3. Reason - Network adapter.
  • Solution - Download and install driver which the system wanted. Better contact your computer technician.
  • 4. Reason - Windows Defender (Windows 8 or 10)
  • Solution - Deactivate windows defender.

Get opened, but doesn't play the video #back to installation menu#back to top

  • 1. Reason - Tick on Remember Playback Password.
  • Solution - ITick on Remember Playback Password.
  • 2. Reason - Maybe the computer has no corresponding decoder.
  • Solution - Download and install K-lite Codec from the link:
Download K-lite Codec

Video opened, but closed suddenly. (suspicious activities detected) #back to installation menu#back to top

  • 1. Reason - Detected suspicious softwears. The TuteOne Video player prevent the user from screen recording.
  • Solution - Uninstall or disable prevented softwears (screen recording softwears etc.)
  • 2. Reason - Skype.exe
  • Solution - Unistall skype or deactivate skype from reading automatically when the system get logged on

Playback password invalid #back to installation menu#back to top

  • 1. Reason - Error in the password.
  • Solution - Please check the password and type carefully, copy and paste is better

Playback password expired. #back to installation menu#back to top

  • 1. Reason - Already used the allowed time period.
  • Solution - I WANT again
  • 2. Reason - Error on System Time and Date
  • Solution - correct the system time and date and reissue the password. (The system time and date must be correct to open the files).
